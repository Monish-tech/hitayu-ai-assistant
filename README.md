# Hitayu AI - Backend (Machine Learning)
The core intelligence system for the Hitayu Health Assistant.

## Technical Overview
- **Machine Learning Model:** Random Forest Classifier (trained for high accuracy).
- **API Framework:** FastAPI for serving predictions.
- **Data Handling:** Processes symptom inputs and maps them to medical diagnoses.

## Key Files
- `disease_prediction.ipynb`: The complete training workflow and data analysis.
- `main.py`: The production-ready API server.
- `disease_prediction_model.pkl`: The serialized ML model.
